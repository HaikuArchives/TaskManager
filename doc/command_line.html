Command Line Options

The following table shows a list of possible command line options:

--help Displays a help text.
--show_deskbar_rep Add replicant to deskbar and then quit.
--hide_deskbar_rep Remove replicant from deskbar and then quit.
--tweak_deskbar Make the deskbar topmost and then quit.
--install Show the installation dialog.
