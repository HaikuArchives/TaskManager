The default plugin couldn't be loaded.

The default plugin 'taskmanager_default.so' couldn't be loaded.
Be TaskManager won't work correctly.
Be TaskManager was looking in this directory for plugins: '<plugin-path>'

This warning is displayed, when the default plugin wasn't found in the plugin path. Be TaskManager depends on this plugin to display the typical system information like CPU and memory usage.

Make sure that the default plugin is installed in the plugins directory and that it is named correctly.
